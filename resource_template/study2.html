Table 2¶

We can also generate a coordinate table from the statistical map.

from nilearn.reporting import get_clusters_table
table = get_clusters_table(zmap, 
                           stat_threshold=2.8,
                           cluster_threshold=10).set_index('Cluster ID',
                                                           drop=True)
display(table)
